# Raj Vincy Degapati - Personal Portfolio

A clean, modern, and interactive personal portfolio website designed to showcase my skills, projects, internships, and achievements as a Front-End Developer and Computer Science Engineering student. 

It is built with a focus on premium aesthetics, featuring dark/light mode toggling, dynamic cursor glows, ambient particle backgrounds, and smooth scroll animations.

![Portfolio Preview](./assets/photo.jpeg) *(Replace this line with an actual screenshot of the live site if desired)*

## ✨ Features

- **Theme Toggling**: Seamless transition between fully realized Light and Dark modes.
- **Interactive UI Elements**: Custom cursor glow trailing the mouse and ambient animated background orbs/particles.
- **Scroll-Telling Intro**: A theme-synced custom intro sequence with strike-through animations.
- **Scroll Animations**: Uses AOS (Animate On Scroll) for revealing elements dynamically as you navigate down the page.
- **Responsive Design**: Designed elegantly for both desktop monitors and mobile devices, ensuring a perfect layout at all breakpoints.
- **Contact Form**: Functional UI layer for a quick email setup directly opening your native mail client.
- **Dynamic Text**: Utilizes Typed.js for an engaging, typing-style subtitle sequence.

## 🛠️ Tech Stack

- **Core**: HTML5, CSS3 (Vanilla), JavaScript
- **Libraries & Assets**: 
  - [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)
  - [Typed.js](https://github.com/mattboldt/typed.js/)
  - [FontAwesome 6.5.0](https://fontawesome.com/) (Icons)
  - [Google Fonts](https://fonts.google.com/) (Inter & Outfit)

## 🚀 Getting Started

To view the project on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Vincyyy07/VINCY_PORTFOLIO.git
   ```

2. **Open the project:**
   Simply open the `index.html` file in any modern web browser.
   *(Alternatively, run a local server like `npx serve .` or use VS Code's "Live Server" extension for a more accurate development experience).*
